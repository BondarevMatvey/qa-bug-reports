Бесконечная загрузка после прохождения авторизации через Apple ID при регистрации

При попытке регистрации нового пользователя на сайте Ozon с использованием авторизации через Apple ID процесс регистрации не завершается корректно.

Окружение:


ОС:Windows 10


Браузер: Google chrome 144.0.7559.13

Шаги воспроизведения

1.Открыть сайт ozon.ru

2.Нажать «Войти» / «Регистрация»

3.Выбрать способ входа через Apple ID

4.Пройти авторизацию Apple (Face ID / пароль)

5.Подтвердить передачу данных

Фактический результат:

После завершения авторизации отображается бесконечный индикатор загрузки.
Переход в личный кабинет не происходит.

Ожидаемый результат

Пользователь должен быть успешно зарегистрирован и автоматически перенаправлен в личный кабинет.

Priority

High — влияет на конверсию и новых пользователей.
